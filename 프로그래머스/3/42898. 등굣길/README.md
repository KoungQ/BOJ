### 문제 설명

계속되는 폭우로 일부 지역이 물에 잠겼습니다.  
철수는 집에서 학교로 가려고 합니다.

집에서 학교까지 가는 길은 **m x n 크기의 격자 모양**으로 나타낼 수 있습니다.  
철수의 집은 **(1,1)**에 있고 학교는 **(m,n)**에 위치합니다.

철수는 **오른쪽 또는 아래쪽으로만 이동**할 수 있습니다.

격자 중 일부에는 **웅덩이(puddles)**가 있어 지나갈 수 없습니다.

집에서 학교까지 갈 수 있는 **경로의 개수**를 구하세요.  
경로의 개수가 매우 클 수 있으므로 **1,000,000,007로 나눈 나머지**를 반환해야 합니다.

### 제한사항

- 격자의 크기 m, n은 각각 **1 이상 100 이하**
- 웅덩이의 개수는 **0개 이상 10개 이하**
- 집과 학교가 있는 위치는 웅덩이가 아닙니다.

### 풀이

이 문제는 **DP (Dynamic Programming)** 문제이다.

`map[y][x]`를 **(1,1)에서 (x,y)까지 오는 경로의 수**라고 정의한다.

한 칸으로 오는 경로는 **위쪽 또는 왼쪽에서만 올 수 있다.**
